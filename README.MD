# 视频号解密

如果你拿到的是本地视频文件，也可以直接解密。

如果不知道如何获取加密URL，可以看github开源项目，想要全自动化一点可以这个文档 https://www.apifox.cn/apidoc/shared-71b9855b-693c-48fc-858b-cde2c5afe5a8/api-34744242
